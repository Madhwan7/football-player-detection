# ⚽ Football Player Detection using YOLO

## 📌 Overview
This project focuses on detecting football players in images and videos using **YOLO (You Only Look Once)** object detection. The model is trained on a custom dataset obtained from Roboflow and uses the **Ultralytics YOLO framework** for efficient training and inference.

---

## 🚀 Features
- Real-time football player detection
- High accuracy using YOLOv8
- Custom dataset integration via Roboflow
- Easy training and inference pipeline
- Visualization using OpenCV

---

## 🛠️ Tech Stack
- Python
- OpenCV
- Ultralytics YOLOv8
- Roboflow
- Jupyter Notebook
